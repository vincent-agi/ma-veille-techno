
# Outils pour la veille technologique

Ce document présente des outils utiles découverts lors de ma veille technologique, avec une courte description et mon avis.

## .NET

- **XSD2Code** ([site officiel](https://www.xsd2code.com/)) : Plugin Visual Studio développé par Pascal Cabanel, permet de générer automatiquement les classes C# à partir d'un fichier XSD (XML Schema). Très utile pour modéliser rapidement des objets complexes et leurs relations.

## Spring Boot

- **Spring Initializr** ([start.spring.io](https://start.spring.io/)) : Générateur de projets Spring Boot. Permet de démarrer rapidement un projet avec les bonnes dépendances et une structure propre.

## TypeScript & Angular

- **QuickType** ([quicktype.io](https://quicktype.io/)) : Générateur de types TypeScript à partir de JSON. Idéal pour éviter l'usage abusif de `any` et garantir un typage strict dans vos applications front-end.

- **Getters and Setters (VSCode)** ([extension](https://marketplace.visualstudio.com/items?itemName=Wilson-Godoi.wg-getters-and-setters)) : Extension VSCode pour générer automatiquement des getters et setters dans vos classes TypeScript. Pratique pour appliquer des transformations ou des validations lors de l'accès aux propriétés.
